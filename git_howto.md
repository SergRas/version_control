# Подсказка по гиту


## Статус репозитория

```sh
git status
```
Создание репозитория:
```sh
git init
```
Добавили в репозитарий файл
```sh
git add
```
Добавили в репозиторий изменения в файле
```sh
git commit -m "Message"
```
Вывод расширенного лога репозитория
```sh
git log
```
Вывод краткого лога
```sh
git log --oneline
```
Переключение между коммитами
```sh
git checkout
```
Создание новой ветки
```sh
git branch "New_name"
```
